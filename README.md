# Ancelin Portfolio

My personal portfolio website built with React, TypeScript, and Vite. It showcases my work, experience, education, and contact information.

## 🚀 Live Demo

[View the live site](https://Anxln.github.io/---------) *(replace after deploying)*

## 🛠️ Tech Stack

- **React 18** + **TypeScript**
- **Vite** (build tool)
- **i18next** (internationalization)
- **CSS** (plain or Tailwind – adjust if needed)

## ✨ Features

- Responsive design (mobile / tablet / desktop)
- Multi‑language support (i18n)
- Dynamic project listing
- Smooth navigation between pages (React Router – assumed)
- Contact form / social links
